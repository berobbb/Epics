# EPICS Project

This repository contains the frontend and the backend code for the Epics Project.

### How to start working?

1. Clone the repository using

   ```
   git clone https://github.com/berobbb/Epics.git
   ```

2. For Frontend -

   ```
   cd client
   ```

3. For Backend -

   ```
   cd server
   ```

4. Install the dependencies

   ```
   npm i
   ```

5. Run the code

   ```
   npm run dev
   ```
